# **App Name**: CartagPay Rewards

## Core Features:

- Offer Display: Show a list of available offers to the user.
- Google Sign-In: Allow users to sign in using their Google account.
- Terms and Privacy Policy Agreement: Require users to agree to the terms and privacy policy before using the app.
- Reward Tracking: Keep track of user progress in completing offers and earning rewards.
- Profile Creation: Store profile details for a specific user using their google details.

## Style Guidelines:

- Primary color: Purple (#9D4EDD) for a modern and engaging feel.
- Background color: Dark gray (#1A1A1A) to provide a sleek, high-tech feel, providing contrast and legibility.
- Accent color: Light purple (#B97DF0) to highlight key UI elements.
- Body and headline font: 'Inter' for a clean and modern aesthetic, working well for both headers and body text.
- Use simple, outlined icons for offers and navigation.
- Use a card-based layout to display offers, making them easy to browse.
- Use subtle transitions and animations to provide feedback when completing tasks.
